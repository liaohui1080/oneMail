著作权归作者所有。 商业转载请联系作者获得授权，非商业转载请注明出处。 作者：Lazlo 链接：http://www.zhihu.com/question/37086510/answer/70751397 来源：知乎 初八日，立冬。每年的这个时候，我都要去莎县吃一碗馄饨。这个习惯已经维持了很多年。每个人都会有一些习惯，也许是因为某个人，也许是因为某件事。在别人看来是怪癖，他却一直在坚持。“每年这个时候，你都会来吃一碗馄饨。我们虽然只见过几面，我却觉得好像认识了你很久。”“你知道混沌和水饺的区别吗？”莎县老板娘问我。我摇头。“饺子越吃越饱，馄饨却会越吃越饿。”我接过她递来的筷子，咬了一口馄饨。二零一五年十一月八号下午三点之前的一分钟我吃了一只馄饨，因为这只馄饨我会记住这一分钟。我想在我今后的人生中，都不会再有机会吃到这么难吃的馄饨。“觉得难吃？”老板娘看着我。“就和当年的我一样”，她的脸上露出追忆的神情，“这条街上曾经有过很多饭店，现在却只剩我一家。为什么会这样，我想了很多年”。“一家店想要开得久，其实味道不重要。这座城市有两千五百万人，一个人赚一块钱，你就已经成功了。明白我的意思吗？”我不太明白。对于数学，我一向很迟钝。不过这不重要。在她说出这番话的那一刻，我的任务已经完成了。我叫拉兹罗，我的职业是替人打探情报，很多年后，我有个绰号叫垃圾篓，不过我不介意，我只不过想让自己过得更好。我告诉老板娘这是我最后一次来莎县。转身离开的那刻，我看见她嘴角浮起一抹微笑。我曾经听人说过，越漂亮的女人越会演戏。我猜她一定很难过。一切都很顺利，只是那年之后，我再也没见过大妈水饺。…………………………………………11.5日更新………………………………………………附赠古龙版↓↓↓↓↓↓↓晌午，有风，微风。阳光透过透明的玻璃照进来，照在莎县古旧的桌子上。阿罗坐在角落里，点了一碗馄饨。他喜欢一个人静静地坐在角落里，享受着这午后独有的悠闲。十五只馄饨。阿罗忽然觉得自己有些寂寞。因为他在数馄饨。当一个人在数馄饨时，那是多么寂寞。“你来了。”老板娘道。阿罗抬起头，眼睛里露出一丝笑意，问道：“你早知道我会来？”老板娘道：“每年的十一月初八，你都会来这里吃一碗馄饨。”阿罗道：”也许今年会是个例外。“老板娘道：”绝不会。“她说得很确定：”说来你也许不信，虽然只见过几面，我却觉得我们好像认识了很久。“阿罗忽然觉得很感动，一个知己，岂不是很多男人梦寐以求的东西？老板娘递给阿罗一双筷子，笑道：”这是我刚煮的馄饨，趁热吃吧。“阿罗掰开筷子，热腾腾的水汽升了起来，抚在他脸上。他心里觉得温暖极了。在冬天里，世上还有什么比吃一碗热腾腾的馄饨更令人畅快的事呢？阿罗咬了一口馄饨。他的脸色突然变了。这只馄饨的皮竟是生的。馄饨馅竟是臭的。阿罗觉得自己的呼吸变得急促，心跳地比平时快了三倍。他从未吃过如此难吃的馄饨。“难吃？”老板娘已经发觉他神色的变化。阿罗沉默。他不愿让别人尴尬，这是他的原则，也是他的美德。阿罗放下筷子，道：“你本可以做得更好。”老板娘道：“不错，我本可以做得更好。”阿罗叹了口气，道：“为什么？”老板娘忽然笑了，笑得很奇怪。她带着笑，对阿罗道：“这条街上曾经有过很多美味的馆子，只不过现在只剩下我这家最难吃的。”阿罗几乎不懂了：“莫非这些食客的脑子都有问题？”老板娘笑道：“据我所知，他们的脑子非但没有毛病，还精明得很。”阿罗敲敲脑袋：“那一定是我的脑子出了毛病。”老板娘笑到：“你的脑子也没有问题。只不过这座城里有两千五百万人，即使他们只来一次，赚的钱也足够维持这家店啦。”
